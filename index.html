Here we go, my notes are now up on Github pages after a complete re-write.



Are you looking to install a better Operating system... one that works 100% of the time, is fast, customisable, stable, satisfying? ...if so, then there's Debian's GNU/Linux and you've found my step-by-step manual on how to get it running with speed.

Linux is free, open-source and powerful... and Debian is possibly the most well tested and supported distro - the basis for Ubuntu and some other distros... it's a very nice OS to get on you PC. Gone are the days of a hanging system! It's time to format those Windoze partitions!

You could test out a Ubuntu distro, like Xubuntu, like I did some years ago. But they come with some bloat and GUI extras. But you could just as well build your own Debian system and get it light-weight and simple.

So why light? My principle is that I might need to run my system on an old or slow machine sometime, and besides that I need free mem available for more demanding tasks, like encoding HD video.

On my Netinstall pages are the steps I take to get Debian up and running using the Debian Net-install CD, which gets the base system installed, retrieving the bulk of it from over the internet. After that, in Steps 2, 3 and 4, I install all the rest that I need. I've kept it pretty minimal, and I'm conscious of choosing low-memory consuming apps where possible.

If you are totally new to Linux, you can easily get Debian running by using a Debian live CD with a full Desktop Environment included. There are links to Live CD's on my Netinstall page.

I hope you'll enjoy Debian - and appreciate too the developers who make it happen!
