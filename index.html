Here we go, my notes are now up on Github pages after a complete re-write.



Looking to install Debian Linux? an OS that works about 100% of the time, is fast, customisable, stable, satisfying ...well if so, you've found my notes on installing and configuring to get it running with speed.

Linux is free, open-source and powerful... and Debian is possibly the most well tested and supported distro - the basis for Ubuntu and some other distros.... Gone are the days of a hanging system! It's time to format those Windoze partitions!

You could test out a Ubuntu distro, like Xubuntu, like I did some years ago. But they come with some bloat and GUI extras. You could just as well build your own Debian system and have it light-weight and simple.

Why light? For me me I need a box that can run every task I need it for with speed. Nothing too fancy.



... The next pages, under Netinstall, will get you started with the install, getting a Net install CD or Live CD from the web..

Using the Debian Net-install CD, gets you the base system installed, retrieving it from over the internet. After that, you can install all the other software that you need. I've kept my package list pretty minimal, and I'm conscious of choosing low-memory consuming apps where possible.

If you are totally new to Linux, you can easily get Debian running by using a Debian live CD with a full Desktop Environment included. There are links to Live CD's on my Netinstall page.

I hope you'll enjoy Debian - and appreciate too the developers who make it happen!
