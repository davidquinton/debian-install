I found the solution to installing the canon driver for this printer at Ubuntu forums (Edgar Ilaga)-- what's needed is to change the .deb package (control file) so that the installation doesn't hang looking for libcupsys2 which is not installed, but uses libcups2 as it should.

  • Download Debian drivers over at http://software.canon-europe.com/products/0010647.asp
  • extract
  • repackage the common .deb.
dpkg-deb -x cnijfilter-common_3.00-1_i386.deb common dpkg-deb --control cnijfilter-common_3.00-1_i386.deb
  • now you see two new folders - common and DEBIAN. now to change the file control in DEBIAN to reflect libcups2.
cd DEBIAN leafpad control
  • look for the line
>Depends: libc6 (>= 2.3.4-1), libcupsys2 (>= 1.2.1), libpopt0 (>= 1.7)
  • and change libcupsys2 to libcups2. Save the file.
  • Now, copy the entire DEBIAN folder into common. then repackage these into a .deb.
dpkg -b common cnijfilter-common_3.00-1_i386.deb
  • install that .deb file
  • Repeat for cnijfilter-ip1900series_3.00-1_i386.deb
dpkg-deb -x cnijfilter-ip1900series_3.00-1_i386.deb common dpkg-deb --control cnijfilter-ip1900series_3.00-1_i386.deb edit the control file as before (change libcupsys2 to libcups2) copy the DEBIAN folder into common, then dpkg -b common cnijfilter-ip1900series_3.00-1_i386.deb
Install that one too, and then, use cups to add the printer: http://localhost:631 (plugged in)
set root ownership of the filter sudo chown root /usr/lib/cups/filter/pstocanonij
