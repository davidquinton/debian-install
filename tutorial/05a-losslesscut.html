Tutorials >
Here's a very handy application using a sleek webkit interface, that can cut up practically any video codec and merge it into a new file without re-encoding.
Get it here



A few things to remember...
  • cuts (coloured areas) will be saved, the blank areas will be lost
  • if you intend to cut out small parts, eg. to make improvements to a film, you will need to set the first cut start point at the very beginning of the movie clip, and its end point at the first part to cut out


Usage

  • bottom < and > to set cut start/end
  • c+ - set each additional cut (it will auto-set the cut start point)
  • nm/am - set auto merge or no merge of cuts
  • click in a cut area to enable reposition of start/end
  • click on cut number to remove it
  • scissors to save the cuts to file (default same dir as source)
  • cd - change save dir


to append movies, use Tools > merge, then load the merged file to make cuts
