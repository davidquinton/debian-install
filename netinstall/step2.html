____Intermediate____


Kdenlive

KDE's non-linear video editor (hence its name.. KDE-n-li-v-e) that is both powerful and easy to use. It's also well supported online.
I give some tips on it on my page, working with HD video.


(Basic install)
If you're not to concerned about getting the latest codecs, you can use this install line - and if you run Debian testing then kdenlive should be quite up to date.

 

Note: since Debian 8 (Jessie), kdenlive depends on libav-tools - no longer ffmpeg.


(Latest release) see my kdenlive page for info on installing the Appimage or from source.


VLMC

from videolan.org (but it looks way difficult to install as vlc is needed first from git source with masses of libs that don't want to install!!)



____Semi_Advanced____


Blender

a 3D animation tool that doubles as video editor, and with professional looks and features...
available in the Debian repo or downloaded from blender.org/..

Its built-in Video Sequence Editor is very good but it can't yet render utilising full CPU power - we need to use a script to split up the work between cpu cores and then concatenate the parts into a complete movie!

See my Blender page for my tips and links to tutorials
