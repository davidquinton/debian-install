I run Debian Jessie without systemd, just because I don't need to go that way - sysvinit boots my system just fine. But it was getting increasingly difficult to install some packages because they depended on systemd, and I needed to install bluefish. This cool html editor however, is dependent on gvfs-backends which depends on libpam-systemd, which depends on the big one, systemd.

Enter Devuan. It looks like they've come a long way developing their Debian fork. They fork anything that depends on systemd to remove the dependency and mirror anything else from the debian repositories - quite a job to be doing.

To migrate to Devuan, at the moment you just need to change your package sources, update, and upgrade and that's all.

  • backup the debian sources.list file
 
  • Download devuan-baseconf from their site http://packages.devuan.org/
  • Then install it with dpkg -i ...
  • enter "jessie" as Devuan suite when asked as it's of course the easiest to migrate to from Jessie (or wheezy)
  • Hit enter with "main"
  • install the devuan keyring and run an upgrade

 


(this upgraded 35 packages and retrieved 4Mb - for a recently updated Jessie install)

Now go ahead and install bluefish or whatever, without any systemd dependency issues!

Download a Devuan iso from their site at https://files.devuan.org/devuan_jessie_beta/
